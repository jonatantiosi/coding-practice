# coding-practice
Contain exercises used for practicing multiple coding languages for university, online courses and/or personal studies as well as codes developed alongside classes.

## python-course
Python 3: From Basic to Advanced
(*Curso de Python 3 do básico ao avançado*)
taught by Luiz Otávio Miranda

### section-3
Fundamentals to programming in Python.

- Python interpreter, modules, comments, and docstrings
- Basic data types: `str`, `int`, `float`, `bool`
- Type coercion and variables
- Arithmetic operators and operator precedence
- String formatting (`f-strings`, `format`, `%`)
- User input and basic I/O
- Conditional statements (`if`, `elif`, `else`)
- Relational and logical operators (`and`, `or`, `not`, `in`)
- Error handling with `try` / `except`
- Loops: `while`, `for`, `break`, `continue`, `else`
- Iterables, iterators, `range`, `enumerate`
- Data structures: `list`, `tuple`
- Mutability vs immutability
- String methods and slicing
- Nested loops and nested data structures
- Basic debugging with VS Code
- Algorithmic exercises (e.g. BMI calculator, secret word game, CPF validation and generator

### section-4
Intermediate Python concepts.

- Defining and using functions (`def`, `return`)
- Positional, keyword, default, positional-only and keyword-only arguments
- Variable scope, namespaces, `global` and `nonlocal`
- `*args` and `**kwargs`
- Higher-order functions and first-class functions
- Closures and functions returning functions
- Dictionaries (`dict`): methods, shallow vs deep copy
- Sets (`set`): operations, methods, and use cases
- List, dict, and set comprehensions
- Lambda functions and functional patterns (`map`, `filter`, `reduce`)
- Iterables, iterators, generator expressions and generator functions
- Error handling: `try`, `except`, `else`, `finally`, `raise`
- Modules and packages (`import`, `__main__`, `__init__.py`, `sys.path`)
- Decorators (basic, with parameters, and execution order)
- Standard library tools (`itertools`, `functools`, `os`, `json`)
- Recursion and recursion limits
- Virtual environments (`venv`) and dependency management (`pip`, `requirements.txt`)
- File handling and context managers (`with open`)
- JSON serialization and persistence
- Common Python pitfalls (mutable default arguments, truthy/falsy values)
- Algorithmic exercises (task list with undo/redo, CPF-related logic, data grouping, list operations)

### section-5 
Introduction to Object-Oriented Programming in Python.

- Classes and objects
- Instance attributes, class attributes, and object state
- Constructors (`__init__`) and object creation
- Instance methods, class methods (`@classmethod`) and static methods (`@staticmethod`)
- The role of `self` and class/method scope
- Encapsulation and naming conventions (`public`, `_protected`, `__private`)
- Relationships between classes: association, aggregation, and composition
- Inheritance (single and multiple) and method overriding
- `super()` and method resolution order (MRO)
- Mixins and code reuse
- Abstract Base Classes (`abc`) and `@abstractmethod`
- Polymorphism and the Liskov Substitution Principle (LSP)
- Custom exceptions and exception handling in OOP
- Special (dunder) methods (`__str__`, `__repr__`, `__new__`, `__call__`, etc.)
- Context managers implemented with classes and `contextlib`
- Function and class decorators
- Metaclasses and class creation lifecycle
- Documentation with docstrings (functions, classes, and modules)
- Enumerations with `enum.Enum`
- Dataclasses (`@dataclass`) and related utilities
- Named tuples and immutable data structures
- Advanced iteration with `Iterable`, `Iterator`, and `Sequence`
- Type hints and modern typing concepts (Python 3.11+ / 3.14+)
- Practical OOP exercises combining abstraction, inheritance, encapsulation, and polymorphism (e.g. banking system domain)

- CSV file processing (`csv.reader`, `csv.writer`, `DictReader`, `DictWriter`)

- Sending emails via SMTP

- Command-line arguments (`sys.argv`, `argparse`)
- HTTP fundamentals and networking concepts
- HTTP servers and client requests (`http.server`, `requests`)
- Web scraping with `requests` and `BeautifulSoup`
- Browser automation with Selenium (WebDriver, waits, element selection)
- Executing external programs with `subprocess`
- Jupyter Notebook usage and experimentation
- Multithreading and parallel execution
- PDF manipulation with `PyPDF2`
- Data structures for queues and stacks (`collections.deque`)
- Excel file manipulation with `openpyxl`
- Image processing with Pillow
